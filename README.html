Before to use


Install


Install php (for development, you can install wamp, it installs php and you'll have a phpmyadmin local page), then install composer

https://getcomposer.org/download/

Then, you have to install all components of composer already used in the project:

Mandatory for all

  • Go to a terminal, in the folder of the project, and copy 'composer install'
  • cd exam_manager
  • composer install

Optional: For development, you can install the symfony server (always in your folder in the terminal):

composer require symfony/web-server-bundle --dev


For local development, run your server with:

php bin/console server:run


Database


Edit this file:

.env.dist

To configure your database, edit this line:

"DATABASE_URL=mysql://db_user:db_password@127.0.0.1:3306/db_name"

with your username and password, and the name of your database.


Create all tables:

delete files in 

src / Migrations
if they exist (exept the gitignore file).

In your terminal, in the folder of the project, copy:

php bin/console doctrine:migrations:diff

and next

php bin/console doctrine:migrations:migrate

Passwords


Change users password

In this project, you have an automatic password when user is create, set to '123', you have to choose another one.

You have to configure it in file:

src / Controller / UserController.php

Edit the line 46 wich set the password to 123.

Next, change the text of the email notification in the file:

src / Utils / Mail.php

Modify two functions: mailNewUser & mailResetPsswrd who informs users of their new password


Change admin password

Admin password is set to 'ExamAdmin123', change it in the file:

config / packages / security.yaml

You can see that all passwords are encoded in bcrypt, and see admin password encoded at line 29.

To edit this password, you have to write it encoded, for this, copy this line in a terminal:

php bin/console security:encode-password

Choose a new password, and copy past the result from the terminal to the file.

templates / base.html.twig
